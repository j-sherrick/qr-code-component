QR Card Demo

Each card in this demonstration showcases a self-contained and reusable Web Component that dynamically generates a QR code based on the URL provided in the `url` attribute. Custom description text can be provided to each element instance through the use of named slots.

QR Cards can be updated dynamically

Use the form to pass in a new URL and description text, and the component dynamically generates a new QR code
